## Magical Toy Overview

* __Extensive Educational Toy Collection:__ Users can browse and explore a wide range of educational toys.
* __User Authentication:__ Users can register and log in to access additional features.
* __CRUD Operations:__ After logging in, users can add, remove, and update their toys using the Create, Read, Update, and Delete (CRUD) operations.
* __Toy Details:__ Users can view detailed information about each toy, including descriptions, images, and specifications.
* __Search Functionality:__ Users can search for specific toys based on keywords or categories.
* __Price Sorting:__ Users can sort the toys by price to find the ones that fit their budget.
